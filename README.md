# Customer-Churn-Prediction
A Machine Learning project that predicts whether a customer is likely to churn based on their demographic and service usage information.

## Features
- Data preprocessing and label encoding
- Exploratory Data Analysis (EDA)
- Model comparison:
  - Logistic Regression
  - Decision Tree
  - Random Forest
  - Gradient Boosting
  - XGBoost
- Hyperparameter tuning using GridSearchCV
- Customer churn prediction through a Streamlit web app

## Tech Stack
- Python
- Pandas
- NumPy
- Scikit-learn
- XGBoost
- Streamlit
- Joblib

## Dataset
IBM Telco Customer Churn Dataset

## Model Performance
- Accuracy: **81%**
- ROC-AUC: **0.86**
